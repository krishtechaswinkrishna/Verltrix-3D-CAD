# Verltrix-3D-CAD
Browser-based 3D CAD studio built with Three.js — sketch entities, extrude/cut/surface/sheet-metal/weldment features, assembly mates, section view, render modes (wireframe/shaded/edges), and analysis overlays, all in a single self-contained HTML file.
Veltrix 3D CAD Studio is a fully client-side 3D CAD environment that runs in any modern browser with no install or server required. Built on Three.js and vanilla JS, it replicates the core SOLIDWORKS workflow — a ribbon CommandManager, a FeatureManager design tree, a PropertyManager panel, and an interactive 3D viewport — across Sketch (rect, circle, polygon, slot, line, spline), Part Features (extrude boss/cut, surface, thicken, shell, fillet, chamfer, hole, rib, weldment, sheet metal, mold, routing), Assembly (components + mates visualized as dashed connectors), and Evaluate modes (stress/flow/zebra-stripe/section-view overlays). The entire app ships as a single Veltrix_3D.html file.
