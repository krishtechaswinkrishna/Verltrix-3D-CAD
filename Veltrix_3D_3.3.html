Quick Start Workflow

  1. Sketch — Draw 2D shapes (Rectangle, Circle, Line, Polygon, Slot, Arc, Spline) on the canvas.
  2. Extrude — Click Features → Extruded Boss/Base (or press E) to turn closed sketch profiles into 3D solids.
  3. Navigate — In the 3D viewport, double-click any face to snap the camera to that face and enter Sketch mode to add more geometry.
  4. Repeat — Sketch on the new face, extrude again. Your full model tree builds up on the left.
  5. Export — Use File → Export STL or Export OBJ to download the model for slicing or rendering.

Sketch Tools

ToolKeyHow to use
Rectangle (RC)RClick and drag to draw a rectangle.
Circle (CI)CClick center, drag to set radius.
Line (LN)LClick start point, click end point. Double-click to finish chain.
Polygon (PG)—Click center, drag for circumscribed polygon (set sides in Inspector).
Slot (SL)—Click first axis point, drag to second axis point.
Arc (AR)—Click center, drag radius, release to set start. Click again for end angle.
Spline (SP)—Click to add control points. Double-click or press Enter to close.
Centerline (CL)—Same as Line but renders dashed — used as revolve axis.

Model Tree

The left panel shows all sketch entities, features, assembly components, and mates. Click any item to select it in the viewport. Hover and click ✕ to delete it (with undo support).

File Operations

Use File → Save to persist your project in the browser (localStorage). Download JSON exports a portable project file. Open loads a previously exported JSON. Export STL / OBJ outputs the 3D model for use in slicers (Cura, PrusaSlicer) or renderers (Blender).
